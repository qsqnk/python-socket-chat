
# Python socket chat  
  
Group chat based on sockets  
  
## Running the project  
  
Clone the project
  
```properties
$ git clone https://github.com/qsqnk/python-socket-chat
```  
Enter the project directory
```properties
$ cd python-socket-chat
```  
Then configure server ip, server port and max message length in `config.py`
Run server
 ```properties
$ python3 server.py
```
